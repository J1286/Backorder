# Backorder System

A web-based backorder management system migrated from Excel.

## Features

- View and edit backorders
- Search orders
- Filter by dealer
- Copy selected orders
- Export to Excel
- Store data in Supabase
- Hosted with GitHub Pages

## Technology

- HTML
- CSS
- JavaScript
- Supabase
- SheetJS (XLSX)

## Deployment

Frontend:
GitHub Pages

Backend:
Supabase

## Notes

This project is intended for internal use.

Do not expose the Supabase service role key.
Only the anon key should be used in the frontend.
